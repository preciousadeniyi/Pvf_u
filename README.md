Implementation of **"The Unified Primitive Valence Foundation: From Solitary Survival to Ecosystemic Negotiation in Affective AI"** (Adeniyi, 2026).

## Overview

PVF-U is a three-tiered architecture for affective AI that:

1. **Guarantees solitary survival** via an Interoceptive Core with epistemic foraging (IG)
2. **Dynamically gates social permeability** via a Relational Gate based on epistemic utility
3. **Implements equitable influence and safety** via an Ecosystemic Superstructure with a Biological Veto

## Installation

### Using Conda (Recommended)

```bash
# Clone the repository
git clone https://github.com/your-username/pvf-u.git
cd pvf-u

# Create conda environment
conda env create -f environment.yml
conda activate pvf-u

# Install in development mode
pip install -e .
